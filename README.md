# ERIC Robotics Dashboard Assignment

## Tech Stack

* React.js
* Vite
* Tailwind CSS

## Features

* Robotics-inspired dashboard UI
* Camera feed switching
* Interactive SLAM-style map
* Path planning visualization
* Radar scanner
* AI detection overlays
* Mission control panel
* ROS-style console logs
* Responsive layout

## Project Structure

The project uses a modular component architecture:

* Sidebar
* Topbar
* MapView
* MiniPreview
* CameraView
* RadarScanner
* MissionPanel
* RosConsole
* AIDetection
* PathPlanner

## Setup Instructions

Clone the repository:

```bash
git clone <your-repo-link>
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```

## Notes

The dashboard simulates robotics telemetry, navigation, and camera systems inspired by modern robotics interfaces.

The implementation focuses on UI fidelity, modularity, and responsive interaction patterns.
